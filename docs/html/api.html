API Reference¶

Deck¶

class card_deck.Deck(cards)¶

A card deck

Parameters

cards (List[Card]) -- A list of cards to start the deck with. These are automatically shuffled

draw(count: int = 1, from_bottom: bool = False)¶

Draw a card from the deck

Parameters

  • count (Optional[int]) -- How many cards to draw from the deck

  • from_bottom (Optional[bool]) -- Whether or not to draw from the bottom

Returns

card(s) -- The card drawn or a list of cards drawn in order

Return type

Union[Card, List[Card]]

insert(card: card_deck.cards.Card, position: int = 0)¶

Insert a card at the given position, zero being the top

Parameters

  • card (Card) -- The card to insert into the deck

  • position (Optional[int]) -- The position at which to insert the card, zero being the top

insert_many(cards: list, position: int = 0)¶

Insert a list of cards at the given position, zero being the top

Parameters

  • cards (List[Card]) -- The list of cards to insert into the deck

  • position (Optional[int]) -- The position at which to insert the cards, zero being the top
